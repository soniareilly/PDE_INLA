# Amortized Hyperparameter Marginalization for PDE Inverse Problems

This repo contains code used for the numerical experiments in https://arxiv.org/abs/2607.03355.

# Installation

This project uses a Conda environment to provide a reproducible installation of FEniCS, hIPPYlib, and the other required Python packages.

Create the environment from the included environment.yml file:

conda env create -f environment.yml

Activate it with:

conda activate amortized
